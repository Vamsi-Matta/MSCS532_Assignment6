# Assignment 6: Medians and Order Statistics & Elementary Data Structures

This repository contains the implementation and analysis for Assignment 6, which focuses on two major topics:

1. Selection algorithms for finding the k-th smallest element in an unsorted array
2. Elementary data structures including arrays, stacks, queues, and linked lists

## Contents

- `selection_algorithms.py`  
  Contains:
  - Deterministic Selection using Median of Medians
  - Randomized Selection using Quickselect

- `data_structures.py`  
  Contains:
  - Array operations
  - Stack implementation
  - Queue implementation
  - Singly Linked List implementation

- `empirical_analysis.py`  
  Runs performance comparisons between deterministic and randomized selection algorithms on different input types and sizes

- `main_demo.py`  
  Demonstrates the functionality of all implemented algorithms and data structures

- `report.docx`  
  Detailed report discussing implementation, complexity analysis, empirical results, and practical applications

## How to Run

Make sure Python 3 is installed on your system.

### Run basic demonstrations

python main_demo.py
